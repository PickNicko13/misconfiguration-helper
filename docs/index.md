# Welcome to MCH

**MCH** (Misconfiguration Scanner) is a powerful command-line tool designed to identify security misconfigurations in networked systems.

## Key Features

- **Port Scanning**: Identifies open TCP ports.
- **Fuzzing**: Detects exposed files and directories.
- **ACAO Scanning**: Checks for CORS misconfigurations.
- **Interactive Acknowledgment**: Mark issues as false positives or resolved.
- **Rich Output**: Colorful and user-friendly console reports.
